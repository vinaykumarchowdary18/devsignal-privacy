# DevSignal HQ — Privacy Policy

**Last updated: July 1, 2026**

## What DevSignal HQ Does

DevSignal HQ is a GitHub App that predicts CI/CD build failures before they run, sending email alerts when a high-risk workflow is detected on your repositories.

## Data We Access

When you install DevSignal HQ, we access:

- **Repository names and metadata** — to identify which repos to monitor
- **GitHub Actions workflow run data** — commit messages, branch names, workflow names, and historical run outcomes (pass/fail)
- **Your GitHub username and email** — to send alert notifications

## How We Use Your Data

We use your data solely to:

- Score incoming workflow runs for failure risk
- Send pre-run email alerts when high risk is detected
- Display your CI prediction dashboard

We do not sell, share, or transfer your data to any third party for any purpose.

## Data Storage

Workflow run metadata and prediction scores are stored in a private database associated with your installation. We retain this data only as long as your installation is active.

## Data Deletion

To delete all your data, uninstall DevSignal HQ from your GitHub account. Your installation data will be permanently deleted within 30 days of uninstallation.
